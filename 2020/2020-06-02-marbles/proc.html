ANOVA

Is there a significant difference in speeed among the marbles?

df_marbles %>%
  mutate(speed = avg_time_lap / track_length_m) %>%
  lm(speed ~ marble_name, data = .) %>%
  anova() %>%
  broom::tidy()

We get nearly the same COD using these features. This is nice because we use fewer DOF for the fit, which will help with the model below:

Now let’s fit on track features along with marble names; this will model the variation in marble lap times, controlling for the track.

df_marbles %>%
  lm(
    formula = avg_time_lap ~ number_laps + track_length_m + marble_name,
    data = .
  ) %>%
  anova() %>%
  broom::tidy()
